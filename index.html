Base Vocabulary

Description of Base Vocabulary

Base Vocabulary is the root for the ontology, as we mentioned above, EA Ontology is mainly developed to describe three kinds of Ethical Assessment. To each one, even though there are similar parts, each one has a different structure, process or point of view to assess some new technology. However, Base Vocabulary have the most important classes in this Ontology as soon as represent the main classes for each Ethical Assessment consider in EA Ontology.

Class Property Description
Ethical Assessment
hasApproach Ethical Assessments
Approach studyOf Approach or method in Ethical Assessment
New Emerging Technology New Technology to be assess
Assessing Expectations subClassOf Method describe by Lucivero
Anticipatory Technology Ethics subClassOf Method describe by Philip Brey
Ethical Impact Assessment subClassOf Method describe in Satori Project
Ethical analysis
Expectation analysisOf
Levels of Ethical Analysis subClassOf
Methods of Ethical Analysis subClassOf

Description of Base Vocabulary in AE
