Base Vocabulary

Description of Base Vocabulary

Base Vocabulary is the root for the ontology, as we mentioned above, EA Ontology is mainly developed to describe three kinds of Ethical Assessment. To each one, even though there are similar parts, each one has a different structure, process or point of view to assess some new technology. However, Base Vocabulary have the most important classes in this Ontology as soon as represent the main classes for each Ethical Assessment consider in EA Ontology.
